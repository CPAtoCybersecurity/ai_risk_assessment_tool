# Conversation Starters / Starter Prompts

> Use the same four starter prompts on every platform that supports them (Microsoft 365 Copilot Agent Builder, Custom GPT, and Gemini Gem all surface starter prompts on the agent's launch screen). Claude Project doesn't have a native starter-prompt field — paste these into the project description or use them as a quick-start cheatsheet.

## The Four Starters

1. **I need to assess a new AI agent before production approval.**
2. **Run the Lethal Trifecta check on a system I'm reviewing.**
3. **Walk me through STRIDE threat modeling for an LLM-based feature.**
4. **Generate a Tier 2 risk assessment report from notes I'll paste.**

## Why these four

- **#1** is the front door — it kicks off the full INTAKE → TRIFECTA → TIER → THREATS → CHECKLIST → RATING → OUTPUT flow.
- **#2** is the fast-path triage. Useful for early-design reviews where the analyst just needs the trifecta verdict.
- **#3** lets a user dive directly into threat modeling for a specific component without going through full intake.
- **#4** is the "I already did the analysis, just write it up" path — paste notes, get a structured Tier 2 document.

## Adapting

If a platform allows more than four (none currently do), add:

5. *Map this AI feature's controls to the NIST CSF 2.0 functions.*
6. *Compare these two AI architectures and tell me which carries more residual risk.*

Do not change the order of #1–#4 — order signals the recommended workflow path to the user.
